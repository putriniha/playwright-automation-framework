# API Testing with Postman

This project demonstrates REST API testing using Postman.

## Tested Endpoints

GET /users
POST /users
PUT /users/{id}
DELETE /users/{id}

## Test Coverage

Positive Test Cases
- Get Users
- Create User
- Update User
- Delete User

Negative Test Cases
- Invalid ID
- Invalid Payload
- Invalid Endpoint

## Validation

- HTTP Status Code Validation
- JSON Response Validation
- Postman Test Scripts (JavaScript)
